Academic Projects/Assignments

I've completed many projects and assignments as a student at Ontario Tech University. Here are some of my achievements:

Database Management System

Designed and implemented a relational database for a library management system. Created tables, relationships, and wrote SQL queries to manage book inventory and user records.

RISC-V Fibonacci Sequence

Designed a program using RISC-V assembly language that recites the fibonacci sequence, up to a specified number times. Created completely from scratch and wrote all code from scratch, using the assembler and runtime simulator, RARS.

Leadership Analysis

Analyzed and studied character "Jean-Luc Picard" from a leadership perspective, and presented my description and analysis. I and other group members chose several characters from fiction that portray strong leadership aspects and laid out our findings in the form of a presentation.

A (Simulated) Pilot Network

Designed a simulated pilot network using Packet Tracer. The pilot network involved setting up multiple network switches, routers, PCs and routing them all together, accross multiple different networks. My group members and I set up and ensured there was full connectivity within the network.
